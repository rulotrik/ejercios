Sentencias en Javascript

Punto y coma

Es recomendable utilizar ; al final de cada sentencia

Podemos escribir varias sentencias en la misma linea separándolas con punto y coma

Espacios en blanco

Los múltiples espacios en blanco se ignoran

Se recomienda utilizar espacios entre operadores =, +, -, *, /

Líneas de más de 80 caracteres

Es recomendable dividir la linea después de un operador

Bloques de código

Se agrupan entre llaves

Se utlizan en funciones, bucles y condiciones, etc ...

Palabras reservadas (keywords)

  • BREAK
  • CONTINUE
  • SWITCH
  • FOR
  • DO ... WHILE
  • IF ... ELSE
  • DEBUGGER
  • TRY ... CATCH
  • FUNCTION
  • RETURNS
